2. [Binary Tree Inorder Traversal](https://leetcode.com/problems/binary-tree-inorder-traversal/)

Given a binary tree, return the inorder traversal of its nodes' values.

Example:

```java
Input: [1,null,2,3]
   1
    \
     2
    /
   3

Output: [1,3,2]
```
